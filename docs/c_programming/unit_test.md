Basic of Unit testing

Very important at EPITECH

Makefile rule for Unit Test

Install Criterion first

```bash
$ gcc -o binary_name file.c tests/test_project.c -- coverage -lcriterion
```

```bash
$ gcovr --exclude tests
```

```bash
$ gcovr --exclude tests --branches
```

```
$ make tests_run
```
